## Stack:
1. TypeScript
2. React
3. React Hooks

## Installation
```
git clone https://github.com/Reyrin/typescript-stopwatch.git
```
## Installing dependencies
```
npm install
```
## Starting a project
```
npm run start
```
## Building a project
```
npm run build
```
